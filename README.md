# Signature Vault

Bringing the world's most iconic restaurant signature dishes to your home kitchen.

## Value Proposition
We provide professional-grade, meticulously tested "copycat" digital recipes and video walkthroughs that capture the exact taste of famous restaurant meals.

## Features
- **Hero Section:** Captures the value proposition of the brand.
- **Recipe Showcase:** High-end display of iconic dishes.
- **Waitlist/Membership:** CTA for "The Vault" membership.
- **Mobile Friendly:** Designed for a seamless experience on any device.

## Tech Stack
- Vite
- React
- Tailwind CSS v4

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
